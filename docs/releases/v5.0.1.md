# WeFlow 5.0.1

## 更新内容

- macOS 自动识别已安装的微信版本。
- 微信 4.1.11 及更新版本不再进入无效的密钥扫描，直接推荐兼容的微信 4.1.8.100。
- 删除“关闭 SIP 并重启 Mac”的引导；安装兼容版微信后重新打开微信即可。
- 欢迎页和设置页增加兼容版本下载与 SHA256 校验入口。
- 应用内官网、GitHub 和自动更新地址切换到项目当前维护入口。
- 新增 [weflow.01mvp.com](https://weflow.01mvp.com)，安装包由 Cloudflare R2 提供备用下载。
