# WeFlow 5.0.10

- 导出默认改为 TXT、全部时间、仅聊天文本，并将批量原始读取并发化。
- 重复导出无变化的会话时复用导出记录，避免重新抽取聊天数据。
- 导出配置改为紧凑布局；完成后可直接打开文件或所在文件夹。
- 修复“我的微信足迹”静默空结果，群聊改用批量文字检索，并加入进度与 10 分钟缓存。
- 统计页不再批量读取头像或媒体文件；优化微信数据库定位，减少启动与分析等待。
- 首次引导新增通用 AI 配置，默认使用 DeepSeek `deepseek-v4-flash` 与 `https://api.deepseek.com`。
