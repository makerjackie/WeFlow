# WeFlow 5.0.2

## 更新内容

- 修复聊天页能浏览消息，但分析页和导出页的消息数显示为 0 的问题。
- 连接数据库时主动初始化消息库索引，并在消息统计前重新确认索引可用。
- 将 WCDB 请求按顺序执行，避免页面同时加载时相互重置数据库状态。
- 作废已经写入的错误零值缓存；分析页也不再信任持久化的零值结果。
