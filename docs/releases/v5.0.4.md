# WeFlow 5.0.4

## 本次更新

- 修复导出页首次统计时可能把有聊天记录的会话显示为 `0` 的问题。
- 总消息数改为一次轻量扫描并优先展示，媒体类型统计只处理当前可见会话。
- 导出统计新增明确的扫描状态和进度条，不再长时间无反馈。
- “我的足迹”新增分阶段真实进度，重复打开同一范围会复用 3 分钟结果缓存。
- “我的足迹”的刷新按钮会跳过缓存，重新读取最新数据。
