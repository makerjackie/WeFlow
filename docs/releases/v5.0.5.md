# WeFlow 5.0.5

## 本次更新

- 修复部分有大量聊天记录的会话在导出时被误判为“没有消息”的问题。
- 原生导出遇到空游标、游标状态失败或 JSONL 数据损坏时，会自动切换兼容读取路径。
- 兼容读取路径可直接扫描实际消息表，不再依赖失效的原生消息库索引。
- 任务中心的导出错误新增复制按钮，错误文字也可直接选中复制。
