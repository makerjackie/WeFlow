# WeFlow 5.0.6

## 本次更新

- 修复兼容导出路径丢失聊天双方身份的问题。
- 微信原始消息表没有直接保存发送者账号时，会通过 `Name2Id` 还原发送者身份。
- 私聊 Markdown 现在会分别标注“我”和联系人名称，不再把全部消息误标成对方。
- 已使用 152,314 条消息的完整会话实测导出，其中双方身份均可正确识别。
