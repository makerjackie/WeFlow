# WeFlow 5.0.7

## 本次更新

- 修复 WCDB 原生游标异常后，兼容读取路径偶发把有消息的会话误判为空的问题。
- 兼容读取改为分别扫描消息表和 `Name2Id`，避免大型关联查询在数据库重连后返回空结果。
- 消息表读取增加一次自动重试；读取失败时不再把不完整结果当成“没有消息”。
- 已用两个会话批量导出实测：共 152,374 条消息，两个会话均成功。
