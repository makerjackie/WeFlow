# WeFlow 5.0.8

## 本次更新

- 修复统计读取失败后被错误显示并缓存为 `0` 的问题；读取不完整时现在会明确提示重试。
- 修复分析页和“我的足迹”在游标读取中断后仍保存不完整结果的问题。
- 修复多个导出任务同时运行时进度串线、一个任务结束后其他任务失去进度的问题。
- 批量导出现在能区分“完成”“部分完成”“已取消”和“失败”，并保留可复制的失败详情。
- 修复消息表部分读取失败仍被当成成功、发送者映射读取失败后继续生成错误身份的问题。
- 原生导出出现截断 JSON 或游标异常时会自动切换兼容引擎，并且只重试失败会话。
- 完善统计缓存失效机制，数据库变化后不再继续使用旧的消息总数和分析结果。

## 验证

- TypeScript 类型检查通过。
- 生产构建和 macOS 应用签名验证通过。
- 使用本机真实微信数据完成消息统计、双会话 Markdown 导出及发送者身份检查。
