# WeFlow 5.0.9

## 本次更新

- 修复 WeLive 返回 `get_messages: no message db` 时没有切换兼容导出引擎的问题。
- 导出页现在只展示真实存在聊天记录的会话；联系人仅用于补充名称与头像，不再把没有消息表的通讯录联系人当成可导出会话。
- 导出页需要统计的对象从完整通讯录收敛到真实会话，减少无效扫描和错误的零消息结果。

## 验证

- TypeScript 类型检查通过。
- 生产构建和 macOS 应用签名验证通过。
- 使用本机真实微信数据完成故障会话的 Markdown 导出：共 3,048 条，双方身份分别为 1,216 条和 1,832 条。
